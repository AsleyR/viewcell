# ViewCell

## Project Description

Next.js application that uses [React Spread Sheet Excel](https://www.npmjs.com/package/react-spread-sheet-excel) to render Excel (XLSX) and CSV files.

## Characteristics

- Open XLSX and CSV Files
- Edit its values
- Change cell colors
- Select cells
- Use formulas
- Undo/Redo
- Merge cells

## Dependencies

- [React Spread Sheet Excel](https://www.npmjs.com/package/react-spread-sheet-excel)
- [XLSX by SheetJS](https://docs.sheetjs.com/docs/getting-started/installation/nodejs/)

## Hosting

The project is hosted on [Vercel](https://vercel.com), and you can access it at: [https://viewcell.vercel.app/](https://viewcell.vercel.app/).
